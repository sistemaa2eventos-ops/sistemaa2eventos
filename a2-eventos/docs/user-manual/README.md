## Introdução
O A2 Eventos é um sistema completo de controle de acesso.

## Acesso ao Sistema
- **Local:** http://localhost:3000
- **Email:** admin@a2eventos.com.br
- **Senha:** (definida no primeiro acesso ou via `reset_admin.js`)

## Dashboard
O dashboard exibe estatísticas em tempo real.

## Gerenciamento de Empresas
### Cadastrar Empresa
1. Clique em EMPRESA
2. Clique em Nova Empresa
3. Preencha os dados
4. Salvar

## Gerenciamento de Funcionários
### Cadastrar Funcionário
1. Clique em FUNCIONÁRIO
2. Clique em Novo Funcionário
3. Preencha os dados
4. Salvar

## Check-in/Check-out
### Check-in Manual
1. Digite CPF ou nome
2. Selecione o funcionário
3. Confirme

### Check-in via QR Code
1. Aponte a câmera
2. Aguarde confirmação

## Monitoramento
Acompanhe acessos em tempo real.
