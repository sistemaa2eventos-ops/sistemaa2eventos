Backend (skeleton)

1. Copie `.env.example` para `.env` e preencha suas chaves do Supabase.
2. Para desenvolver sem Docker: `npm install` e `npm run dev`.
3. O backend expõe rotas básicas: `/health`, `/api/auth/*`, `/api/public/*`.

Conectar ao Supabase
- Use `reboot/supabase/schema.sql` para criar as tabelas iniciais no seu projeto Supabase.
